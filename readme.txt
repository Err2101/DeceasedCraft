# Make sure you have Java 17 installed as 1.18.2 server requires Java 17 to run the server.
# https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html
#
# Windows Installation:
# Run the forge installer 1.18.2-40.2.4 and click Install server, then select this directory as the server base.
# After that, launch the run.bat file to start the server, and make sure to agree with eula on the first startup.
#
# Linux Installation:
# While in the DeceasedCraft directory, run `java -jar forge-1.18.2-40.2.4-installer.jar --installServer` to install forge.
# After that, `run chmod +x run.sh` to make the run script executable.
# Now start run.sh with bash (`bash run.sh`) to start the server, and make sure to agree with eula on the first startup.
